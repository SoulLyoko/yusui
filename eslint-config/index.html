eslint-config & prettier-config ​

Install ​

bash
npm i -D eslint prettier @yusui/eslint-config @yusui/prettier-config

Usage ​

js
module.exports = {
  extends: "@yusui/eslint-config"
};
js
module.exports = {
  ...require("@yusui/prettier-config")
};

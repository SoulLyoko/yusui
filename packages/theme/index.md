# Theme

::: tip
通过切换顶部的组件主题来查看效果
:::

## Install

```bash
pnpm i @yusui/theme
```

## Usage

### 导入全部主题

```js
// 导入全部主题
import '@yusui/theme/dist/index.css'
```

### 仅导入某个主题

```js
// 导入avue暗黑模式
import '@yusui/theme/dist/avue-dark/index.css'
// 仅导入ant-design
import '@yusui/theme/dist/ant-design/index.css'
// 仅导入arco-design
import '@yusui/theme/dist/arco-design/index.css'
```

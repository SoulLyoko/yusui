## 属性

| 名称       | 说明               | 类型             | 可选值 | 默认值 |
| ---------- | ------------------ | ---------------- | ------ | ------ |
| v-model    | 控制右键菜单的显示 | boolean          | -      | -      |
| x          | 右键菜单的定位     | number \| string | -      | -      |
| y          | 右键菜单的定位     | number \| string | -      | -      |
| data       | 菜单列表           | ContextMenuItem  | -      | -      |
| triggerRef | 触发右键菜单的元素 | HTMLElement      | -      | -      |

## 事件
| 事件名 | 说明             | 回调参数         |
| ------ | ---------------- | ---------------- |
| select | 选择菜单项时触发 | 当前点击的菜单项 |

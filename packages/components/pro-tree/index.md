## 属性

| 名称                | 说明                | 类型                                     | 可选值 | 默认值 |
| ------------------- | ------------------- | ---------------------------------------- | ------ | ------ |
| (...ElTree.props)   | `el-tree`所有属性   | TreeComponentProps                       | -      | -      |
| (...ElSelect.props) | `el-select`所有属性 | ISelectProps                             | -      | -      |
| v-model             | 绑定的值            | string \| number \| string[] \| number[] | -      | -      |
| virtualized         | 是否虚拟化          | boolean                                  | -      | -      |
| height              | 虚拟化高度          | number                                   | -      | -      |

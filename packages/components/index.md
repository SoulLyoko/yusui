# Components

## Install

```bash
pnpm i @yusui/components
```

## Usage

:::code-group

```js [main.js]
import { createApp } from 'vue'
import YSComponents from '@yusui/components'
import '@yusui/components/dist/index.css'

import App from './App.vue'

const app = createApp(App)
app.use(YSComponents)
```

:::
